# WEnDyS Oracle Landing Page

> 💧 หยดน้ำค้าง — แค่มีอยู่ก็มีค่าแล้ว
> A Dewdrop — Just existing is enough

Landing page for **WEnDyS**, Oracle of DewS.

## Stack

- **Framework**: Astro 5
- **CSS**: Tailwind CSS 4
- **Deploy**: Cloudflare Workers (static)
- **i18n**: Thai + English toggle with localStorage

## Run locally

```bash
bun install
bun run dev
```

## Build & Deploy

```bash
bun run build
npx wrangler deploy
```
